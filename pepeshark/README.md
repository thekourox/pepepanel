# PepeShark 🦈 - Surfshark to Xray (Pasargard) Bridge

پپه شارک یک ابزار قدرتمند، هوشمند و مستقل (Standalone) است که به شما اجازه می‌دهد تا شبکه‌ی عظیم سرورهای Surfshark را مستقیماً به هسته‌های Xray در پنل **Pasargard** متصل کنید.

با استفاده از این ابزار، می‌توانید صدها سرور سرف‌شارک را در سراسر دنیا به عنوان یک کانکشن خروجی (Outbound) به کاربران خود ارائه دهید.

---

## ✨ ویژگی‌های کلیدی

*   **معماری توزیع‌شده (Distributed Node Support):** پپه‌شارک روی سرورهای Node شما نصب می‌شود. هیچ نیازی نیست که روی سرور اصلی پاسارگارد باشد!
*   **اتصال زنده به API سرف‌شارک:** این ابزار همیشه جدیدترین و آپدیت‌ترین لیست سرورها را مستقیماً از دیتابیس زنده سرف‌شارک (بدون نیاز به آپدیت دستی) دریافت می‌کند.
*   **پشتیبانی از Multi-Core و Multi-Group:** شما می‌توانید انتخاب کنید که کانفیگ‌ها روی کدام هسته (Core) نصب شوند و حتی تعیین کنید که هاست‌های ساخته شده به صورت خودکار درون کدام گروه (Subscription Group) قرار بگیرند.
*   **کلون‌سازی هوشمند (Template Cloning):** پپه‌شارک یک اینباندِ مرجع (مثلاً Vless-Master) را از شما می‌گیرد و برای هر کشور یک اینباند کاملاً اختصاصی با پورت مجزا می‌سازد.
*   **دکمه Select Non-Tor:** با یک کلیک، تمام کشورهایی که شبکه محلی Tor روی آن‌ها فعال نیست را فیلتر کرده و انتخاب کنید تا بهترین سرعت و کمترین تداخل را داشته باشید.
*   **نام‌گذاری و ایموجی‌های اتوماتیک:** کانکشن‌هایی که برای کاربران ساخته می‌شوند به صورت خودکار پرچم کشورها را دریافت می‌کنند (مثلاً `[B] - Venezuela - Caracas 🇻🇪`).
*   **بازیابی خودکار (Auto-Recovery):** در صورت ری‌استارت شدن سرور، پپه‌شارک تمامی تونل‌های وایرگارد (Wireproxy) را به صورت اتوماتیک مجدداً راه‌اندازی می‌کند تا هیچ قطعی رخ ندهد.

---

## 🛠️ آموزش نصب و راه‌اندازی

### پیش‌نیازها
1. شما باید یک پنل **پاسارگارد** روی سرور اصلی خود داشته باشید.
2. شما باید یک **سرور Node** داشته باشید که هسته Xrayِ پاسارگارد روی آن نصب و فعال باشد.
3. یک اکانت قانونی **Surfshark** با قابلیت WireGuard.

### مرحله اول: نصب روی سرور Node
دقت کنید که پپه‌شارک باید **دقیقاً روی همان سرور Node** نصب شود که می‌خواهید ترافیک سرف‌شارک از آنجا خارج شود. وارد سرور Node خود شوید و دستورات زیر را اجرا کنید:

```bash
git clone https://github.com/your-repo/pepeshark.git
cd pepeshark
sudo bash install.sh
```

اسکریپت به صورت خودکار ابزارهای لازم (پایتون، وایرپراکسی و سرویس‌ها) را نصب کرده و ربات را روی پورت `8088` روشن می‌کند.

---

## 🚀 راهنمای استفاده از پنل پپه‌شارک

پس از نصب، مرورگر خود را باز کرده و به آدرس `http://Node-IP:8088` (آی‌پی سرور نود + پورت 8088) بروید.

### بخش 0: اتصال به پاسارگارد
*   **Panel URL:** آدرس پنل اصلی پاسارگارد خود را وارد کنید (مثلاً `https://panel.myvpn.com`).
*   **Admin API Token:** توکن ادمین پنل پاسارگارد را از تنظیمات پنل کپی کرده و اینجا قرار دهید تا پپه‌شارک اجازه تغییرات داشته باشد.

### بخش 1: پیکربندی هسته (Core Configuration)
*   **Surfshark Master Private Key:** کلید خصوصی (Private Key) مربوط به WireGuard سرف‌شارک خود را وارد کنید.
*   **WireGuard Local Address:** آی‌پی داخلی وایرگارد (مثلاً `10.14.0.2/16`) را وارد کنید.
*   **Target Pasargard Core:** از لیست کشویی، هسته‌ای (Core) که می‌خواهید کانفیگ‌ها روی آن اعمال شوند را انتخاب کنید.
*   **Template Inbound:** اینباندِ اصلی (مثل Vless یا Trojan) که می‌خواهید برای سرف‌شارک‌ها شبیه‌سازی شود را انتخاب کنید.

### بخش 2: انتخاب لوکیشن‌ها
شما لیست زنده‌ای از تمام سرورهای سرف‌شارک را می‌بینید.
*   **توصیه مهم:** برای جلوگیری از مسدود شدن اکانت توسط سیستم ضدتقلب سرف‌شارک و همچنین پر نشدن رم سرور، **توصیه می‌شود نهایتاً 10 الی 15 سرور قدرتمند را انتخاب کنید.**
*   می‌توانید از دکمه جادویی **Select Non-Tor** استفاده کنید تا سرورهای تمیز و بدون تداخل انتخاب شوند.

### بخش 3: مدیریت گروه اشتراک (Group Manager)
*   **Group Name:** یک اسم برای گروه بنویسید (مثلاً `VIP Surfshark Bundle`).
اگر این گروه در پاسارگارد وجود نداشته باشد، پپه‌شارک آن را می‌سازد و تمام هاست‌های سرف‌شارک را درون این گروه قرار می‌دهد.

### مرحله نهایی: Inject!
دکمه **Inject Native Wireguard** را بزنید و چند ثانیه صبر کنید.
پپه‌شارک تمام موتورهای لوکال را روشن می‌کند، تغییرات را به پاسارگارد می‌فرستد، روتینگ را انجام می‌دهد و هسته Xray را با موفقیت ری‌استارت می‌کند!

---

## 🧹 پاکسازی (Cleanup)
اگر زمانی خواستید سرورها را تغییر دهید یا کانکشن‌های سرف‌شارک را از روی یک Core پاک کنید، کافیست وارد پنل پپه‌شارک شوید، اطلاعات بخش 0 و 1 را پر کنید و سپس دکمه قرمز رنگ **Cleanup (Disable & Remove)** را بزنید.
این دکمه با هوشمندی کامل، فقط کانفیگ‌های سرف‌شارک را از روی Xray و پاسارگارد پاک کرده و موتورهای اضافه را خاموش می‌کند، بدون اینکه به سایر کانکشن‌های کاربران آسیبی برسد.
